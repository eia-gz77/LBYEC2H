\subsection{How were the objectives achieved?}
By:
\begin{enumerate}
\item Writing mesh equations for a 4-loop resistive circuit and converting them into Matrix form Ai = c.
\item Implementing Matlab scripts to calculate determinants for Cramer's rule and inverses for matrix solution.
\end{enumerate}
	
\subsection{What are the key results and generalizations?}
The key results are:
\begin{enumerate}
\item The mesh currents were consistently calculated to be i1 = 0.0344A, i2 = -0.0160A, i3 = -0.0413A, and i4 = -0.0867A across all methods.
\item The eigenvalues of the system matrix were calculated as approximately 184.05, 145.95, 130.60, and 45.40.
\end{enumerate}
